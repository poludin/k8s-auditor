# 🛡️ k8s-auditor

**k8s-auditor** — это легковесная CLI-утилита, написанная на Go, для автоматизированного аудита Kubernetes-кластеров. 

Инструмент проверяет инфраструктуру на соответствие 14 правилам "Золотого стандарта", находит "мусорные" ресурсы, оставшиеся после настройки кластера, и помогает поддерживать архитектурную чистоту без ручного вмешательства.

---

## 🎯 Ключевые возможности (Roadmap)

- [x] **Подключение к K8s API**: Безопасная работа через локальный `kubeconfig`.
- [ ] **Поиск осиротевших ресурсов**: Обнаружение неиспользуемых сервисов (nginx, registry) и забытых bootstrap-нод.
- [ ] **Валидация манифестов**: Проверка наличия обязательных лейблов и аннотаций.
- [ ] **Анализ ConfigMaps**: Поиск хардкода и проверка логики формирования URL-адресов (учет суффиксов окружений, например, `dev`).
- [ ] **Авто-исправление (Fix)**: Автоматическое приведение неймспейсов к эталону "Золотого стандарта".

## 🚀 Быстрый старт

### Требования
* Установленный Go (версия 1.20+)
* Доступ к Kubernetes кластеру
* Настроенный файл `~/.kube/config`

### Установка (Сборка из исходников)

Склонируйте репозиторий и соберите бинарный файл:

```bash
git clone https://github.com/poludin/k8s-auditor.git
cd k8s-auditor
go build -o k8s-auditor main.go
```

### Использование

Запуск базового сканирования кластера:

```bash
./k8s-auditor scan
```

Вы можете явно указать путь к конфигурационному файлу кластера:

```bash
./k8s-auditor scan --kubeconfig=/path/to/custom/kubeconfig
```

## 🏗️ Архитектура

Проект построен с использованием следующих технологий:

- **Go** — основной язык разработки.
- **Cobra (spf13/cobra)** — фреймворк для построения CLI-интерфейса.
- **client-go** — официальный клиент для взаимодействия с Kubernetes API.
